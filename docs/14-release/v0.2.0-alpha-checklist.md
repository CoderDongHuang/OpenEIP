## Full-stack Acceptance

发布冒烟必须通过以下真实容器边界：

1. Gateway、Frontend、Java、Python 和 MySQL 健康。
2. 用户注册、登录、`/me` 与非管理员 RBAC 拒绝。
3. 文件上传、下载摘要一致、知识库创建与文档关联。
4. OCR、解析、Embedding、RAG 内部 API 共用同一身份和请求上下文。
5. Java Chat SSE 经 Python 返回 grounded answer 和 Citation，并持久化消息历史。
6. Java Agent SSE 执行显式授权的 `knowledge.search`，产生工具生命周期和完成事件。
7. 删除关联、知识库和文件后资源不可再访问。
