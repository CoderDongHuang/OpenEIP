## Known Limitations

- 尚未交付 Auth、RAG、Agent、Workflow、Connector 等业务能力。
- Kafka、Milvus、LLM Streaming、MCP 等技术选择仍需 Phase 1.5 Spike。
- Foundation 以中文文档为规范源；英文同步版在稳定版本前补齐。
- Docusaurus 上游开发依赖仍报告 MODERATE 问题，但没有 HIGH/CRITICAL，且不进入运行时镜像。

## Publish Procedure

1. 提交全部 Foundation 文件并创建 Pull Request。
2. 确认远程 CI 全部通过。
3. 合并到 `main`。
4. 创建并推送 `v0.1.0-alpha` 标签。
5. Release Workflow 重新运行 CI、发布 GHCR 镜像并创建 GitHub Release。
