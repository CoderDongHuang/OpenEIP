# Issue #45: MVP OCR Engine

> GitHub Issue: [#45](https://github.com/CoderDongHuang/OpenEIP/issues/45)
> Milestone: `v0.2.0 MVP`
> Status: Approved for Implementation

The accepted MVP scope is safe PNG/JPEG decoding, internal caller authentication and identity context,
a replaceable OCR provider port, a deterministic bounded raster provider, normalized blocks/results,
tests, benchmark, security review, and compatibility evidence. Production PaddleOCR models, handwriting,
rotation, complex layouts, table extraction, and v0.3 language/accuracy claims are explicitly outside
#45.
