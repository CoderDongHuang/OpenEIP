# OpenEIP MVP

OpenEIP 是面向企业场景规划的开源 AI 平台。当前版本为 `v0.2.0-alpha`，提供可验证的单节点 MVP；默认确定性 AI Provider 用于离线验证，不代表生产模型能力。

## 当前交付

- Auth/RBAC、文件与知识库控制面
- OCR、解析、Embedding 和 grounded RAG 流水线
- 流式 Chat 工作区与受约束 Agent Runtime
- Docker Compose 单节点验证环境
- PRD、SAD、模块 SDD、RFC、ADR、API 和 SPI 契约

完整设计文档位于仓库的 `docs/` 目录。
