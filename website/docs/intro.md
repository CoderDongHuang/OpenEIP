# OpenEIP Foundation

OpenEIP 是面向企业场景规划的开源 AI 平台。当前版本为 `v0.1.0-alpha`，提供工程治理、架构基线和可运行脚手架，不包含后续路线图中的业务能力。

## 当前交付

- Java 21 + Spring Boot 基础服务
- Python 3.12 + FastAPI 基础服务
- React + Vite 前端壳
- Docker Compose 开发环境
- PRD、SAD、SDD、RFC 和 ADR 流程

完整设计文档位于仓库的 `docs/` 目录。
