# Security Policy

## 报告安全漏洞

如果你发现安全漏洞，**请不要在公开 Issue 中报告**。

请通过以下渠道私密报告：

- GitHub Security Advisory："Report a vulnerability"
- 详细流程参见 [docs/00-governance/security-policy.md](docs/00-governance/security-policy.md)

## 响应 SLA

| 严重程度 | 初始响应 |
|---|---|
| CRITICAL | 24 小时内 |
| HIGH | 48 小时内 |
| MEDIUM | 1 周内 |
| LOW | 2 周内 |
